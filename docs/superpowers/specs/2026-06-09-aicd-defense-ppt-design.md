# AI 创作者辅助生产与分发平台项目答辩 PPT 设计规格

## 目标

为当前项目制作一份 20-25 页的项目答辩 PPT，实际规划为 23 页，预计答辩时长 12-15 分钟。PPT 使用 `D:\Browserdownload\产品介绍.pptx` 作为视觉模板参考，面向大型互联网公司字节跳动的答辩老师，风格为科技感、正式、工程可信。

## 叙事主线

采用平台级全链路答辩方案，并在关键页注入技术架构深挖和内容平台治理视角。

叙事顺序：

1. 产品价值开场：解释 AIGC 创作、内容治理和分发闭环的业务价值。
2. 工程架构证明：展示前后端分层、AI 服务封装、数据模型、离线同步、Redis 热榜等系统设计。
3. 内容平台治理贴合字节：突出内容安全、质量评分、推荐排序、运营后台和多平台分发。
4. 工程交付与演进收束：说明测试、部署、评估、边界和未来路线。

## 视觉方向

- 参照模板的版式体系、字号层级、页面装饰、页脚/页码风格。
- 用深色科技感章节页承接关键段落，用浅色信息页呈现架构、流程、表格和结论。
- 每页必须有可视元素，避免纯文本堆叠。
- 技术页优先使用架构图、流程图、公式卡片、数据模型关系图、能力矩阵。
- 整体表达要正式、克制、专业，避免过度花哨。

## 页数与节奏

- 总页数：23 页。
- 答辩时长：12-15 分钟。
- 普通页面讲解 30-45 秒。
- 架构、发布流水线、内容治理等重点页讲解 45-60 秒。
- 每页包含备注，备注按半演讲稿形式撰写，每页约 80-150 字，重点页可更详细。
